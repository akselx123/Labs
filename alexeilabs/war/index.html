Практическое задание 5

Задание на взаимодействие классов. Необходимо было придумать ряд классов со своими методами и переменными и организовать взаимодействие между ними.

Разработанные классы:

  • Класс Освещение {переменные: bool on, sensor sen; методы: включить если сработал датчик, выключить, получить состояние}
  • Класс Датчик {переменные: bool on; методы: включить, выключить, получить состояние}
  • Класс Термометр {переменные: double temp; методы: поднять температуру, понизить температуру, получить состояние}
  • Класс Отопление {переменные: bool on; методы: включить используя температуру термометра, выключить, получить состояние}
  • Класс Кондиционер {переменные: bool on; методы: включить используя температуру термометра и если не включено отопление, выключить, получить состояние}
  • Класс Сирена {переменные: bool on, String logs; методы: включить если постовлена сигнализация и сработал датчик, выключить, получить состояние, получить историю срабатывания, отчистить иторию}
  • Класс Сигранизация {переменные: bool on, String pass; методы: включить используя введенный пароль, выключить используя введенный пароль, получить стстояние}

Описание взаимодействия: Если срабатывает датяик движения включается свет, если выставленная температура ниже чем показывает термометр, то включается кондиционер, если заданная температура выше чам по термометру, то включается отопление. Сигнализация включится если введен перный пароль. После включения сигнализации отключаютсч все приборы. Сирена сработает если включена сигнализация и сработал датчик.
Задание консольное и просмотра нет =(

Практическое задание 5. Исключения

Работа с исключениями. Есть несколько полей для ввода. В одно можно вводить только числа, в другое только e-mail, в третье нельзя вводить ковычки и т.д.
Все это сделано с помощью механизма исключений. То есть когда возникает неверная ситуация (ex: введено не число), то создается пользовательское исключение и авдается сообщение о ошибке.

Практическое задание 6

Работа с CSV файлами. С сайта открытых данных были скачаны два файла в csv формате. Один файл содержит маршруты движения городского транспорта. Второй файл содержит Государственные регистрационные знаки маршрутных транспортных средств, движение которых разрешено по выделенной полосе.
Обработка данных:

  • Вывести все маршруты городского транспорта
  • Вывести все данные о ГРЗ и марках авто
  • Вывести наименования имеющихся перевозчиков
  • Вывести маршрут по введенному ГРЗ авто (выводит конечные точки маршрута из файла 1)
  • Вывести маршрут и марки автобусов по введенному номеру автобуса (Выводит маршрут из файла 1 и все найденные марки авто из файла 2)
  • Вывести маршрут и марки автобусов по введенному перевозчику (выводит все авто, принадлежащие перевозчику из файла 2 и маршруты номеров транспорта, принадлежащих перевозчику из файла 1)
